# Java Graduate Program - Java Elite 🎓

Este repositório contém o desenvolvimento técnico, exercícios e projetos práticos realizados durante a pós-graduação **Java Elite (Anhanguera/Cogna)**. O foco do programa é a especialização avançada em ecossistema Java, Arquitetura de Sistemas e Integração com IA.

## 🚀 Principais Tecnologias Exploradas
*   **Linguagem & Core:** Java 21+, Stream API, Concurrency, Virtual Threads.
*   **Frameworks:** Spring Boot 3, Quarkus, Langchain4j.
*   **Arquitetura:** Clean Architecture, DDD, Event-Driven (Kafka/RabbitMQ).
*   **Infra & Cloud:** Docker, Kubernetes, AWS (S3, Lambda).
*   **IA Corporativa:** RAG (Retrieval Augmented Generation), Agentes Inteligentes, MCP.

## 📂 Conteúdo do Programa

### 1. Fundamentos & Concorrência
Exploração profunda de APIs de I/O, Reflection, Collections e o novo modelo de Virtual Threads para alta performance.

### 2. Back-end & AI Native
Desenvolvimento de APIs resilientes com Spring e Quarkus, integrando LLMs através de Langchain4j e bancos vetoriais.

### 3. Arquitetura e System Design
Aplicação de Design Patterns (Gang of Four) e estratégias de escalabilidade horizontal, particionamento de dados e Teorema CAP.

### 4. Qualidade e Testes
Implementação de pirâmide de testes com JUnit, Mockito, Testcontainers e análise de métricas via Sonar.
